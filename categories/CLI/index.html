Get football information in your terminal with football

By CIAvash • May 12, 2016

App::Football is a command line program I wrote in Raku. football lets you access information regarding football(soccer!) teams, leagues, tables, fixtures, scores and players. It uses another module I wrote called WebService::FootballData which is a Raku interface for football-data.org API.
